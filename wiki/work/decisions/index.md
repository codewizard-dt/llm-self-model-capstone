# Decisions — Active Items

Lists **only active** Decision Groups — a group stays listed while at least one of its decisions is `proposed`. When every block is `accepted` or `superseded`, delete the group's line here — the file itself never moves; per-decision status lives in each block. See the [lifecycle](lifecycle.md).

Entry format: `- [DEC-NNNN — Title](DEC-NNNN-slug.md) — one-line summary · D1 proposed, D2 accepted, …`
