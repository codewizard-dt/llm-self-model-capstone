# Giacomo Rancurel

relates_to::[[Agentic Physical Autoresearch]]
relates_to::[[Imitation Learning — ACT]]
uses::[[LeRobot]]
uses::[[Flywheel]]
uses::[[SO101 Arm]]

GitHub: giacomoran | X/Twitter: @ludocomito

Lead developer of the **hyperfamila** project at HackRome 2026 — a hobbyist-scale agentic physical autoresearch system built in 8 hours. The project demonstrated the full loop (data collection → policy training → autonomous agent control) with an SO101 arm, custom syringe end-effector, ACT policies, and OpenAI Codex.

**Awards**: Best use of Codex (OpenAI), Best use of Flywheel (Paradigma), $1k grant (UnitdVentures), Best Demo (Apeira).

**Open source**: all code and dataset are public.
- Code: https://github.com/giacomoran/hyperfamila-hackrome-2026
- Dataset: https://huggingface.co/datasets/giacomoran/hyperfamila_provette

**Potential collaboration**: the hyperfamila stack is the closest public reference implementation for the ENPIRE-inspired capstone experiment. Reaching out to @ludocomito / @giacomoran is a reasonable step before building from scratch.
