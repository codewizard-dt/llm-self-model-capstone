## Reframe the Problem

Instead of building physical robots:

Build a **robot evolution simulator**.

Input:

```

Move across rough terrain as quickly as possible.
```

The factory:

- Generates robot designs
- Simulates them
- Measures performance
- Mutates successful designs
- Repeats

After 100 generations:

```
Generation 1: 12m traveled
Generation 10: 31m traveled
Generation 50: 89m traveled
Generation 100: 142m traveled
```

Now you're demonstrating:

- AI agents
- Evolutionary search
- Optimization
- Emergent behavior
- Autonomous design

without touching hardware.
