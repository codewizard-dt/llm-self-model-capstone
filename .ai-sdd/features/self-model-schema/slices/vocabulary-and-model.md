## `contracts/vocabulary.py`

One `StrEnum` per axis, values seeded from MASTER_REQUIREMENTS → Parts Catalog Grammar:

- `MotorAllocation`: `2drive+1arm+1claw`, `2drive+2free`, `4drive`, `3drive+1manip`
- `ArmPosition`: `front`, `rear`, `side`, `absent`
- `EndEffector`: `claw_grasper`, `bare_arm`, `none`
- `WheelConfig`: `front_omni+rear_standard`
- `ArmGearRatio`: `7:1`, `1:1`
- `Cartridge`: `200rpm`

This is the single source of truth F3 (parts-catalog-grammar) will later import to generate `parts_catalog.json` — do not duplicate these value sets anywhere else (D1, C1).

## `contracts/self_model.py`

`SelfModel` (pydantic v2, `extra="allow"`), reusing F1's `StrictModel` base where a strict sub-model is wanted:

- `schema_version: Literal["1.0"]` (default `"1.0"`, locked) · `generation: int` (≥0) · `parent_generation: int | None`
- `model_validator`: gen 0 ⇒ `parent_generation is None`; gen > 0 ⇒ `parent_generation` is an int and `< generation`
- `config: SelfModelConfig` — six axes typed against the `vocabulary` enums; `extra="forbid"`
- `structural: StructuralLayer` — `parts: list[Part]` (`Part{id:str, type:str}`), `connections: list[Connection]` (`Connection{source:str, target:str, kind:str}`); `Part`/`Connection` use `extra="allow"`; empty lists allowed (edge fields are `source`/`target`, not `from`/`to`)
- `capability: CapabilityLayer` — `reach_mm`, `max_grip_force_N`, `max_pull_force_N`, `com_height_mm` (float, default 0); `extra="forbid"`
- `predictive: dict[str, dict[str, float | bool | str]]` — free task-string keys
- `gap_model: dict[str, dict[str, float]]` — free task-string keys; reuses F1's `gap` residual keys per task
- `reasoning: str` — `min_length=1`

Re-export `SelfModel` and the vocabulary enums from `contracts/__init__.py` (keep all F1 exports).

## `make schema` + freeze gate (D8)

`make schema` regenerates **both** `schemas/contract_line.json` (unchanged output) and `schemas/self_model.json` from the models — single source of truth is the pydantic models. Add a freshness gate (e.g. a `schema-check` target, or fold into `validate`) that runs `make schema` then `git diff --exit-code schemas/`, asserting committed == generated. Emit mechanism is your choice provided `contract_line.json` stays byte-stable (the gate enforces this).
