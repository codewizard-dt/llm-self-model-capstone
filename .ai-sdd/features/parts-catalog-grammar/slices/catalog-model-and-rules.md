## `contracts/parts_catalog.py`

- **`PartsCatalog`** (pydantic v2, `extra="forbid"`): `schema_version: Literal["1.0"]` (default `"1.0"`, locked) + one `list[<Enum>]` field per axis (`motor_allocation`, `arm_position`, `end_effector`, `wheel_config`, `arm_gear_ratio`, `cartridge`), each typed against the **imported** `contracts.vocabulary` enums (D1 — never redefine the value sets) + a typed `constraints: CatalogConstraints` sub-model holding the machine-readable rule set (D4).
- **`Violation`** = `{code: str, message: str}`; **`CatalogVerdict`** = `{buildable: bool, violations: list[Violation]}` (D9).
- **`validate_config(config: SelfModelConfig) -> CatalogVerdict`** — deterministic combinatorial-buildability engine (reuses F2's `SelfModelConfig`, D3). Rules:
  - **R1 (D7)** `end_effector == claw_grasper` ⇒ `motor_allocation == 2drive+1arm+1claw` (claw needs arm-lift + claw motor = 2 manipulator motors). Else → violation `code="CLAW_MOTOR_BUDGET"`.
  - **R2** `scoop`/`flywheel` need ≥1 manipulator motor; all three remaining allocations supply ≥1, so both are valid on each (`4drive` is gone at the vocabulary level — `s0`/D6).
  - **R3 (D8, forward-only)** `end_effector == flywheel` ⇒ `cartridge == 600rpm`. Else → violation `code="FLYWHEEL_CARTRIDGE"`. Whether `600rpm` suits a claw/scoop is **not** judged here (F9's torque call) — claw/scoop accept any cartridge.
  - Each violation carries a human-readable `message` (e.g. `"end_effector 'flywheel' requires cartridge '600rpm'; got '200rpm'"`).

## `contracts/catalog.py` + `make catalog` / `catalog-check`

Mirror F2's as-shipped `schema.py` pattern (a registry + `write_text`, **not** a stdout redirect): `catalog.py`'s `main()` writes `contracts/parts_catalog.json` directly. The six axis arrays are generated **from the vocabulary enum members** (their values, in declared order) so the file can never drift from the vocabulary; enrich with `schema_version` and the machine-readable `constraints` block (D4). `parts_catalog.json` is a *data* document at the contracts root — **not** a JSON Schema, and no schema is emitted for `PartsCatalog` (D5).

- `catalog: $(UV) python -m contracts.catalog` (no redirect).
- `catalog-check: catalog` then `git diff --exit-code parts_catalog.json` (asserts committed == generated; mirrors `schema-check`).
