## `contracts/src/contracts/control_command.py`

- **Imports** — `from __future__ import annotations`; `from typing import Annotated, Literal, Union`; `from enum import StrEnum`; `from pydantic import BaseModel, ConfigDict, Field, model_validator`; `from contracts.motor_telemetry import StrictModel`.
- **Clamp constants (module-level, exported)**:
  ```python
  MAX_LINEAR = 0.35           # m/s   — matches pros_bridge/src/main.cpp current literal
  MAX_OMEGA  = 0.60           # rad/s — matches pros_bridge/src/main.cpp current literal
  MAX_FLYWHEEL_RPM = 3600.0   # output shaft RPM — proposed default; F20 may narrow
  ARM_DEG_MIN = 0.0
  ARM_DEG_MAX = 360.0         # conservative; F20 narrows per assembled build
  TTL_MS_MAX  = 1000          # matches bridge.py's ttl_ms clamp
  ```
- **`FaultCode(StrEnum)`** — exactly these 8 values, in this order: `"malformed_json"`, `"unknown_command"`, `"ttl_expired"`, `"watchdog"`, `"estop_latched"`, `"out_of_range"`, `"oversized_packet"`, `"not_assembled"`.
- **`ControlEnvelope(StrictModel)`** — `extra="forbid"`:
  - `v: Literal[1] = 1`
  - `seq: int = Field(ge=0, lt=2**32)`
  - `sent_ms: int = Field(ge=0)`
  - `ttl_ms: int = Field(ge=1, le=TTL_MS_MAX)`
  *(Note: `type` is the outer discriminator; placed on the wire-line models below, not on the envelope, so the discriminated union routes cleanly.)*
- **Command bodies** — each subclasses `ControlEnvelope` and adds `type: Literal["cmd"]` + a `cmd` literal + body fields. All `extra="forbid"`:
  - `StopCommand`: `type: Literal["cmd"] = "cmd"`, `cmd: Literal["stop"] = "stop"`, `reason: Literal["operator_estop", "operator", "watchdog_self", "fault"]`.
  - `DriveCommand`: `cmd: Literal["drive"] = "drive"`, `vx: float = Field(ge=-MAX_LINEAR, le=MAX_LINEAR)`, `vy: float = Field(default=0.0, ge=0.0, le=0.0)` (locked to 0 — D12), `omega: float = Field(ge=-MAX_OMEGA, le=MAX_OMEGA)`.
  - `TurnCommand`: `cmd: Literal["turn"] = "turn"`, `omega: float = Field(ge=-MAX_OMEGA, le=MAX_OMEGA)`.
  - `ArmCommand`: `cmd: Literal["arm"] = "arm"`, `deg: float = Field(ge=ARM_DEG_MIN, le=ARM_DEG_MAX)`, `vel_rpm: float | None = None`.
  - `ClawCommand`: `cmd: Literal["claw"] = "claw"`, `state: Literal["open", "close"]`, `grip_force_N: float | None = None`.
  - `FlywheelCommand`: `cmd: Literal["flywheel"] = "flywheel"`, `rpm: float = Field(ge=0.0, le=MAX_FLYWHEEL_RPM)`.
- **`ControlCommand`** — the inner discriminated union:
  ```python
  ControlCommand = Annotated[
      Union[StopCommand, DriveCommand, TurnCommand, ArmCommand, ClawCommand, FlywheelCommand],
      Field(discriminator="cmd"),
  ]
  ```
- **`HeartbeatLine(ControlEnvelope)`** — `type: Literal["heartbeat"] = "heartbeat"`, `extra="forbid"`. Envelope-only (no `cmd`).
- **`ControlLine`** — outer discriminated union over `type` of (`ControlCommand` for `type=="cmd"`, `HeartbeatLine` for `type=="heartbeat"`):
  ```python
  ControlLine = Annotated[
      Union[StopCommand, DriveCommand, TurnCommand, ArmCommand, ClawCommand, FlywheelCommand, HeartbeatLine],
      Field(discriminator="type"),
  ]
  ```
  *(Implementation note: because the 6 command bodies all carry `type: Literal["cmd"]`, pydantic's `type`-discriminator routes to `ControlCommand` which then routes by `cmd`. If pydantic v2 requires a two-step adapter to express this cleanly, wrap the 6 command bodies under a thin `_CommandLine` model whose `type` is `"cmd"` and whose inner discriminator is `cmd` — preserve the wire shape exactly.)*
- **`AckLine(StrictModel)`** — `extra="forbid"`:
  - `v: Literal[1] = 1`
  - `type: Literal["ack"] = "ack"`
  - `ack: int = Field(ge=0, lt=2**32)` — the `seq` it acknowledges
  - `state: Literal["ok", "rejected", "stale"]`
  - `recv_ms: int = Field(ge=0)`
  - `fault: FaultCode | None = None`
  - **`@model_validator(mode="after")`**: enforce `fault is None` iff `state == "ok"`; `state == "rejected"` requires non-null `fault`; `state == "stale"` accepts either (typical: `fault = "ttl_expired"`).

## Acceptance

1. `make sync` exits 0.
2. `make lint` exits 0 (`ruff check` + `ruff format --check`).
3. `make test` exits 0, covering at minimum these cases in `tests/test_control_command.py`:
   - **Round-trip parse**: one valid instance of each of the 7 line shapes (6 commands + 1 heartbeat + 1 ack) via `pydantic.TypeAdapter(ControlLine)` / `AckLine.model_validate`.
   - **Discriminated union**:
     - `cmd: "fly"` (out of vocabulary) → `ValidationError`;
     - `type: "telemetry"` → `ValidationError` (closed `Literal["cmd","heartbeat"]` on `ControlLine`);
     - missing `cmd` field on a `type="cmd"` line → `ValidationError`;
     - `cmd: "drive"` missing `vx` → `ValidationError`.
   - **Range bounds**:
     - `drive.vx > MAX_LINEAR` and `drive.vx < -MAX_LINEAR` → `ValidationError`;
     - `drive.vy != 0` → `ValidationError` (D12);
     - `drive.omega > MAX_OMEGA` and `< -MAX_OMEGA` → `ValidationError`;
     - `turn.omega < -MAX_OMEGA` → `ValidationError`;
     - `arm.deg < ARM_DEG_MIN` and `> ARM_DEG_MAX` → `ValidationError`;
     - `flywheel.rpm < 0` and `> MAX_FLYWHEEL_RPM` → `ValidationError`;
     - `ttl_ms < 1` and `> TTL_MS_MAX` → `ValidationError`.
   - **Envelope shape**:
     - `v != 1` → `ValidationError`;
     - any unknown extra field on any model → `ValidationError` (`extra="forbid"`);
     - `seq < 0` and `seq >= 2**32` → `ValidationError`.
   - **Ack invariants**:
     - `AckLine(state="ok", fault="ttl_expired", ...)` → `ValidationError`;
     - `AckLine(state="rejected", fault=None, ...)` → `ValidationError`;
     - `AckLine(state="stale", fault="ttl_expired", ...)` → valid;
     - `AckLine(state="ok", fault=None, ...)` → valid.
   - **Clamp-constant identity** — import each constant from `contracts` and assert equality with the documented values bit-for-bit.
   - **Cross-contract demux (D2)** — load one line from `contracts/fixtures/session_example.jsonl`; assert it parses cleanly through `ContractLine` **and** fails to parse as `ControlLine` (the `type` discriminator is mutually exclusive). Proves the single-stream demux is wire-level safe before any fixture is added.
4. `make schema` exits 0 and writes `contracts/schemas/control_command.json` + `contracts/schemas/ack_line.json` (committed, non-empty; the `type`/`cmd`/`state`/`fault` discriminators surface as JSON-Schema `enum` arrays). `make schema-check` exits 0; `contract_line.json`, `self_model.json`, `score_contract.json` are **byte-identical** to their committed pre-F19 versions.
5. `from contracts import ControlLine, ControlCommand, AckLine, FaultCode, MAX_LINEAR, MAX_OMEGA, MAX_FLYWHEEL_RPM, ARM_DEG_MIN, ARM_DEG_MAX, TTL_MS_MAX` (and every other re-exported name) resolves in `uv run python` after `make sync`.
